# Simulador de Aplicação Financeira em C++

Projeto acadêmico desenvolvido em C++ para simular aplicações financeiras com juros compostos.

## Funcionalidades

- Escolha entre três tipos de investimento
- Validação de capital mínimo e prazo mínimo
- Cálculo de juros compostos
- Exibição do montante final formatado

## Conceitos utilizados

- Estruturas condicionais (if/else, switch)
- Estruturas de repetição (while, do-while, for)
- Validação de entrada de dados
- Formatação de saída
